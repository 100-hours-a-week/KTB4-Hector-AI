# 회고
이번 주 전까지는 머신러닝은 단순히 기계 학습, 딥러닝은 신경망 기반의 학습 정도로만 알고 있었다. 딥러닝이 비교적 최근의 기술이고 AI라는 큰 틀안에 여러가지 방법중 하나 정도로만 알고있었다.
어떤 구조를 가지고 있는지 어떻게 동작하는지 알고는 있었지만 코드를 실제 구현해보거나 내부를 상세하게 본적은 없었다. 이번 위클리 챌린지를 진행하면서 직접 코드를 작성하고 모델을 짜보았다.
처음 코드를 받아 적을 때는 받아쓰기 느낌이 나서 의미가 있는건지 싶었지만 평일 내내 같은 코드를 반복적으로 작성하고 실행시키다보니 교안을 보고 작성해야했던 코드들도 혼자서 할 수 있었다.
복습을 진행하면서 AI의 도움을 받아서 진행했었던 학부 시절 과제와 달리 시간도 오래 걸리고 성능도 비교적 떨어지지만 이제 누가 task를 줘도 모델을 선정하고 코드를 작성할 수 있을 것 같다.
