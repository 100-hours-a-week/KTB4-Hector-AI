# KTB4-Hector-AI
# How to use
```
python main.py start --name "이름"
```
# 사용 예시
## 첫 실행
이름을 입력했을 때 이름이 없다면 User정보를 입력할 User.json과 개인 단어 저장 json파일을 만듭니다.
이 후 비밀번호를 설정하고 원하는 단어를 입력합니다.
후에 다시 접속 했을 때 비밀번호가 정확하지 않다면 다시 입력해야합니다.
## 메뉴 1번
1번을 누르면 단어와 단어의 의미를 삽입할 수 있습니다.
## 메뉴 2번
2번을 누르면 단어 테스트를 할 수 있습니다. 단어는 랜덤으로 생성되며 오답 시 정답을 알려주고 마지막에 총 문제에서 맞춘 문제 개수를 알려줍니다
## 종료
메뉴 3번을 누르면 종료 됩니다.
